# UCB1 の詳細証明（gap-dependent 上界）

このノートでは、UCB1 の標準的なリグレット上界を、定数込みで最後まで追います。
[01-証明パターン.md](01-証明パターン.md) が「型」を掴むためのノートだとすると、こちらは「実際に最後まで書き切る」ためのノートです。

## 目標

劣最適腕 $i$ ごとに

```math
\mathbb{E}[N_i(T)]
\leq
\frac{8 \log T}{\Delta_i^2} + 1 + \frac{\pi^2}{3}
```

を示し、そこから

```math
R_T
\leq
\sum_{i:\Delta_i>0}
\left(
\frac{8 \log T}{\Delta_i}
+
\left(1 + \frac{\pi^2}{3}\right)\Delta_i
\right)
```

を導きます。

## 設定

- 腕は $i = 1, \dots, K$
- 報酬は各腕ごとに i.i.d.
- 報酬は $[0,1]$ に入る
- 腕 $i$ の平均報酬は $\mu_i$
- 最適平均は $\mu^* = \max_i \mu_i$
- 劣最適腕のギャップは $\Delta_i = \mu^* - \mu_i > 0$
- 時刻 $t$ までに腕 $i$ を引いた回数を $N_i(t)$ と書く
- 腕 $i$ を $s$ 回引いたときの経験平均を $\hat{\mu}_i(s)$ と書く

初期化として、最初に各腕を 1 回ずつ引くとします。

## UCB1

時刻 $t \geq K+1$ で

```math
I_t \in \arg\max_i
\left[
\hat{\mu}_i(N_i(t-1)) + \sqrt{\frac{2 \log t}{N_i(t-1)}}
\right]
```

とします。

信頼半径を

```math
\mathrm{rad}(s,t) := \sqrt{\frac{2 \log t}{s}}
```

と書けば、UCB 指標は

```math
\mathrm{UCB}_i(t)
=
\hat{\mu}_i(N_i(t-1)) + \mathrm{rad}(N_i(t-1), t)
```

です。

## リグレット分解

累積リグレットは

```math
\begin{aligned}
R_T
&=
\sum_{t=1}^T (\mu^* - \mu_{I_t}) \\
&=
\sum_{i:\Delta_i>0} \Delta_i \, \mathbb{E}[N_i(T)].
\end{aligned}
```

したがって、各劣最適腕 $i$ について $\mathbb{E}[N_i(T)]$ を抑えれば十分です。

## 固定した劣最適腕を数える

以下、劣最適腕 $i$ を一つ固定します。

任意の整数 $\ell \geq 1$ に対して

```math
N_i(T)
\leq
\ell
+
\sum_{t=K+1}^T
\mathbf{1}\{ I_t = i \text{ and } N_i(t-1) \geq \ell \}
```

です。

意味は単純で、腕 $i$ が引かれる回数は

- 最初の $\ell$ 回
- それ以降に、なおも選ばれた回数

に分けられる、ということです。

## 核心: 劣最適腕が選ばれたなら何が起きたか

時刻 $t$ に腕 $i$ が選ばれ、さらに $N_i(t-1) \geq \ell$ だとします。

このとき

- $s := N_{i^*}(t-1)$
- $s_i := N_i(t-1)$

とおけば、選択規則より

```math
\hat{\mu}_{i^*}(s) + \mathrm{rad}(s,t)
\leq
\hat{\mu}_i(s_i) + \mathrm{rad}(s_i,t)
```

が成り立ちます。

ここで、つぎの三つの事象を考えます。

```math
A_t(s):=
\left\{
\hat{\mu}_{i^*}(s)
\leq
\mu^* - \mathrm{rad}(s,t)
\right\}
```

```math
B_t(s_i):=
\left\{
\hat{\mu}_i(s_i)
\geq
\mu_i + \mathrm{rad}(s_i,t)
\right\}
```

```math
C_t(s_i):=
\left\{
\mu^*
<
\mu_i + 2 \mathrm{rad}(s_i,t)
\right\}
```

### なぜ三つのどれかが起きるのか

もし $A_t(s)$ も $B_t(s_i)$ も $C_t(s_i)$ も起きていないとすると、

```math
\hat{\mu}_{i^*}(s) > \mu^* - \mathrm{rad}(s,t)
```

なので

```math
\hat{\mu}_{i^*}(s) + \mathrm{rad}(s,t) > \mu^*
```

です。

一方、$B_t(s_i)$ と $C_t(s_i)$ が起きていなければ

```math
\hat{\mu}_i(s_i)
<
\mu_i + \mathrm{rad}(s_i,t)
```

かつ

```math
\mu_i + 2 \mathrm{rad}(s_i,t) \leq \mu^*
```

なので

```math
\hat{\mu}_i(s_i) + \mathrm{rad}(s_i,t) < \mu^*
```

となります。

すると

```math
\hat{\mu}_{i^*}(s) + \mathrm{rad}(s,t)
>
\hat{\mu}_i(s_i) + \mathrm{rad}(s_i,t)
```

となって、腕 $i$ が選ばれたことに矛盾します。

したがって、腕 $i$ が選ばれたなら、少なくとも $A_t(s)$, $B_t(s_i)$, $C_t(s_i)$ のどれか一つは起きています。

## 第三の事象を消す

ここで

```math
\ell := \left\lceil \frac{8 \log T}{\Delta_i^2} \right\rceil
```

と置きます。

すると $s_i \geq \ell$ かつ $t \leq T$ のとき

```math
2 \mathrm{rad}(s_i,t)
\leq
2 \sqrt{\frac{2 \log T}{\ell}}
\leq
\Delta_i
```

です。

よって

```math
\mu_i + 2 \mathrm{rad}(s_i,t) \leq \mu_i + \Delta_i = \mu^*
```

となり、$C_t(s_i)$ は起こりえません。

これが、係数 $8$ が現れる場所です。
必要だった条件は

```math
2 \sqrt{\frac{2 \log T}{\ell}} \leq \Delta_i
```

であり、両辺を二乗すると

```math
\ell \geq \frac{8 \log T}{\Delta_i^2}
```

を得ます。

## 期待値を評価する

以上から、

```math
\mathbf{1}\{ I_t = i \text{ and } N_i(t-1) \geq \ell \}
```

は、ある $s \in \{1,\dots,t-1\}$ と $s_i \in \{\ell,\dots,t-1\}$ について
$A_t(s)$ または $B_t(s_i)$ が起きることによりしか 1 になりません。

したがって

```math
\begin{aligned}
\mathbb{E}[N_i(T)]
&\leq
\ell
+
\sum_{t=K+1}^T
\sum_{s=1}^{t-1}
\sum_{s_i=\ell}^{t-1}
\mathbb{P}\!\left( A_t(s) \cup B_t(s_i) \right).
\end{aligned}
```

合併界より

```math
\mathbb{P}\!\left( A_t(s) \cup B_t(s_i) \right)
\leq
\mathbb{P}(A_t(s)) + \mathbb{P}(B_t(s_i)).
```

Hoeffding 不等式から

```math
\mathbb{P}(A_t(s)) \leq t^{-4},
\qquad
\mathbb{P}(B_t(s_i)) \leq t^{-4}
```

なので

```math
\mathbb{P}\!\left( A_t(s) \cup B_t(s_i) \right)
\leq
2 t^{-4}.
```

ゆえに

```math
\begin{aligned}
\mathbb{E}[N_i(T)]
&\leq
\ell
+
\sum_{t=K+1}^T
\sum_{s=1}^{t-1}
\sum_{s_i=\ell}^{t-1}
2 t^{-4} \\
&\leq
\ell
+
\sum_{t=1}^{\infty} 2 t^{-2}.
\end{aligned}
```

最後の不等式では、
$\sum_{s=1}^{t-1} \sum_{s_i=\ell}^{t-1} 1 \leq t^2$ を使っています。

さらに

```math
\sum_{t=1}^{\infty} \frac{1}{t^2} = \frac{\pi^2}{6}
```

より

```math
\sum_{t=1}^{\infty} 2 t^{-2} = \frac{\pi^2}{3}.
```

したがって

```math
\mathbb{E}[N_i(T)]
\leq
\ell + \frac{\pi^2}{3}
\leq
\frac{8 \log T}{\Delta_i^2} + 1 + \frac{\pi^2}{3}.
```

## リグレット上界

以上をリグレット分解に代入すると

```math
\begin{aligned}
R_T
&=
\sum_{i:\Delta_i>0} \Delta_i \, \mathbb{E}[N_i(T)] \\
&\leq
\sum_{i:\Delta_i>0}
\left(
\frac{8 \log T}{\Delta_i}
+
\left(1 + \frac{\pi^2}{3}\right)\Delta_i
\right).
\end{aligned}
```

これが UCB1 の標準的な gap-dependent 上界です。

## この証明のどこが学習上重要か

- 「良い事象を大きく一つにまとめる」のではなく、実際に比較される二つの腕だけを見る
- 劣最適腕が選ばれたときの事象を三つに分解する
- 第三の事象を閾値 $\ell$ の選び方で潰す
- 残りを Hoeffding 不等式で機械的に足し上げる

この四点が見えていれば、UCB1 の証明はかなり再現しやすくなります。
