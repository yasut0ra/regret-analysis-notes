# 下界の演習問題

このノートは、下界の基礎を読んだあとに、自分で内容を整理するための演習集です。
完全な解答は書かず、短いヒントだけ載せています。

## 使い方

- まず [04-下界の基礎.md](04-下界の基礎.md) を読む
- 次に [05-KLとchange-of-measure.md](05-KLとchange-of-measure.md) を読む
- そのあと、この演習を紙に書いて解く

## 問題 1. 上界と下界の違いを説明する

バンディットにおける上界と下界の違いを、自分の言葉で 4 行から 8 行で説明してください。

最低限、つぎの語を使ってください。

- あるアルゴリズム
- 任意のアルゴリズム
- 問題の難しさ

ヒント:
「このアルゴリズムは良い」と「誰がやってもここまでは unavoidable」の違いを意識すると整理しやすいです。

## 問題 2. リグレットと試行回数の関係を確認する

リグレット分解

```math
R_T
=
\sum_{i:\Delta_i>0} \Delta_i \, N_i(T)
```

から、なぜ下界の議論で $\mathbb{E}[N_i(T)]$ を下から抑えることが重要なのか説明してください。

ヒント:
各劣最適腕について「何回は引かざるをえないか」が分かれば、それを $\Delta_i$ 倍して足せます。

## 問題 3. instance-dependent と minimax を区別する

つぎの二つの主張の違いを説明してください。

```math
\liminf_{T \to \infty}
\frac{\mathbb{E}_{\nu}[R_T]}{\log T}
\geq
\sum_{i:\Delta_i>0}
\frac{\Delta_i}{d(\mu_i,\mu^*)}
```

```math
\inf_A \sup_{\nu} \mathbb{E}_{\nu}[R_T(A)]
\geq
c \sqrt{KT}
```

ヒント:
一つ目は「固定した問題」、二つ目は「最悪の問題」です。

## 問題 4. KL の役割を説明する

下界で KL ダイバージェンスが自然に現れる理由を、数式をあまり使わずに説明してください。

ヒント:
「一回観測すると、どれくらい二つの世界を見分ける情報が増えるか」という言い方が有効です。

## 問題 5. Bernoulli KL を計算する

Bernoulli$(p)$ と Bernoulli$(q)$ の KL ダイバージェンス

```math
d(p,q)
=
p \log \frac{p}{q}
+
(1-p)\log\frac{1-p}{1-q}
```

に対して、$p=0.4$, $q=0.6$ を代入した式を書いてください。
数値計算までは不要です。

ヒント:
定義にそのまま代入するだけです。

## 問題 6. 小さいギャップでの見え方を確認する

$q = p+\Delta$ で $\Delta$ が小さいとき

```math
d(p,p+\Delta) \approx \frac{\Delta^2}{2p(1-p)}
```

という近似が成り立つとします。

ここから

```math
\frac{\Delta}{d(p,p+\Delta)}
```

がなぜ $1/\Delta$ オーダーになるのか説明してください。

ヒント:
$d(p,p+\Delta)$ が $\Delta^2$ に比例するなら、その逆数は $1/\Delta^2$ に比例します。

## 問題 7. 別世界を作る意味を説明する

なぜ下界の証明では、元の世界 $\nu$ と、最適腕が違う別世界 $\nu'$ を作るのでしょうか。

ヒント:
アルゴリズムに「二つの世界で違う行動をしないと困る」状況を作りたいからです。

## 問題 8. 事象 $E = \{ N_i(T) > T/2 \}$ の意味

固定した劣最適腕 $i$ に対して

```math
E := \{ N_i(T) > T/2 \}
```

という事象を考える理由を説明してください。

ヒント:
元の世界では起きてほしくなく、別世界では起きてほしい事象になっています。

## 問題 9. 腕を一つだけ変える利点

なぜ別世界 $\nu'$ を作るときに、腕 $i$ だけを変えると議論が簡単になるのでしょうか。

ヒント:
change-of-measure の左辺に出る和が、ほぼ一項だけに縮みます。

## 問題 10. $\log T$ が出る理由を説明する

下界で $\log T$ が現れる理由を、誤差確率や「区別に必要な情報量」という言葉を使って説明してください。

ヒント:
時間が長いほど、間違えないことへの要求が厳しくなります。

## 問題 11. 上界と下界を見比べる

UCB1 の上界

```math
\sum_{i:\Delta_i>0} \frac{\log T}{\Delta_i}
```

と、Bernoulli bandit の instance-dependent 下界

```math
\sum_{i:\Delta_i>0}
\frac{\Delta_i \log T}{d(\mu_i,\mu^*)}
```

を見比べて、「なぜ両者はオーダーとして矛盾していないのか」を説明してください。

ヒント:
小さいギャップでは $d(\mu_i,\mu^*)$ が $\Delta_i^2$ に比例します。

## 問題 12. 自分の言葉で下界の証明骨格を書く

数式を見ずに、下界の証明の流れを文章だけで 5 行から 10 行で説明してください。

最低限、つぎの語を使ってください。

- 別世界
- 事象
- KL ダイバージェンス
- pull 数
- 情報量

ヒント:
「どんな世界を作って、何を比べて、どこから pull 数が出るか」の順に書くとまとまりやすいです。
