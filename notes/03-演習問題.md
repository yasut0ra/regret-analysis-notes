このノートは、UCB 系のリグレット解析を「読める」だけでなく、「自分で再現できる」ようにするための演習集です。
完全な解答は書かず、短いヒントだけ載せています。

- 次に [02-UCB1-詳細証明.md](02-UCB1-詳細証明.md) を読んで標準証明を追う
- そのあと、この演習を紙に書いて解く

## 問題 1. リグレット分解を導く

時刻 $t$ に選んだ腕を $I_t$、時刻 $T$ までに腕 $i$ を引いた回数を $N_i(T)$ とする。

つぎを示してください。

```math
R_T
=
\sum_{t=1}^T (\mu^* - \mu_{I_t})
=
\sum_{i:\Delta_i>0} \Delta_i \, \mathbb{E}[N_i(T)].
```

ヒント:
$\mathbf{1}\{ I_t = i \}$ を使って、時刻方向の和を腕方向の和に入れ替えます。

## 問題 2. 良い事象の確率を評価する

[01-証明パターン.md](01-証明パターン.md) の良い事象

```math
E_t
:=
\bigcap_{i=1}^K
\bigcap_{s=1}^{t}
\left\{
\left| \hat{\mu}_i(s) - \mu_i \right|
\leq
\mathrm{rad}(s,t)
\right\}
```

について、Hoeffding 不等式と合併界から

```math
\mathbb{P}(E_t^c) \leq 2K t^{-3}
```

を導いてください。

ヒント:
まず片側確率を $t^{-4}$ で抑え、左右二つを足してから、さらに全腕・全試行回数にわたって足します。

## 問題 3. 三つの事象への分解を確認する

[02-UCB1-詳細証明.md](02-UCB1-詳細証明.md) で使った三つの事象

- $A_t(s)$
- $B_t(s_i)$
- $C_t(s_i)$

について、もしこれらが全て起きていないなら、なぜ腕 $i$ は選ばれえないのかを示してください。

ヒント:
最適腕の UCB は $\mu^*$ より大きく、劣最適腕の UCB は $\mu^*$ より小さいことを順に示します。

## 問題 4. 係数 8 はどこから来るか

UCB1 では

```math
\mathrm{rad}(s,t) = \sqrt{\frac{2 \log t}{s}}
```

なので、第三の事象を消すには

```math
2 \mathrm{rad}(s_i,t) \leq \Delta_i
```

が必要です。

ここから

```math
\ell \geq \frac{8 \log T}{\Delta_i^2}
```

が出ることを確認してください。

ヒント:
まず $t \leq T$ を使って $\log t$ を $\log T$ に置き換え、次に両辺を二乗します。

## 問題 5. 標準証明の定数を確認する

[02-UCB1-詳細証明.md](02-UCB1-詳細証明.md) の最後では

```math
\sum_{t=1}^{\infty} 2 t^{-2} = \frac{\pi^2}{3}
```

を使っています。

なぜ三重和が最終的に $2 \sum_t t^{-2}$ で抑えられるのかを説明してください。

ヒント:
時刻 $t$ を固定したとき、$s$ と $s_i$ の取り方の数は高々 $t^2$ 個です。

## 問題 6. 半径の定数を変えてみる

信頼半径を

```math
\mathrm{rad}_{\alpha}(s,t) := \sqrt{\frac{\alpha \log t}{2s}}
```

とします。

このとき、同じ流れで解析した場合の閾値 $\ell_i$ が
$\alpha$ にどう依存するかを求めてください。

ヒント:
必要条件は

```math
2 \mathrm{rad}_{\alpha}(\ell_i, T) \leq \Delta_i
```

です。

## 問題 7. gap-dependent 上界の弱点を説明する

つぎの上界

```math
R_T
\leq
\sum_{i:\Delta_i>0}
\left(
\frac{8 \log T}{\Delta_i}
+
\left(1 + \frac{\pi^2}{3}\right)\Delta_i
\right)
```

が、$\Delta_i$ が非常に小さい問題ではなぜ扱いにくいのかを説明してください。

ヒント:
$1 / \Delta_i$ の項に注目すると、最適腕とほぼ同じ平均を持つ腕がいると上界が大きくなります。

数式を見ずに、UCB1 の解析の流れを文章だけで 5 行から 10 行で説明してください。

- 劣最適腕
